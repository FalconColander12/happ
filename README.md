# Назначение приложения

**Happ** — программное кроссплатформенное решение, предназначенное для формирования, настройки и управления пользовательскими профилями подключения.

Приложение использует технологии на основе **Xray** и предоставляет локальную среду для работы с конфигурациями соединения на мобильных устройствах и настольных компьютерах. Пользователь может самостоятельно создавать профили, добавлять подготовленные настройки, изменять параметры подключения, сохранять несколько вариантов конфигураций и выбирать необходимый набор параметров для различных сценариев использования.

Ключевым элементом архитектуры Happ является система отдельных конфигурационных профилей. Каждый профиль представляет собой независимый набор данных, включающий параметры соединения, правила маршрутизации, настройки обработки сетевых потоков и дополнительные управляющие элементы.

Такая модель позволяет хранить множество конфигураций одновременно, редактировать каждый профиль отдельно и быстро переключаться между сохранёнными вариантами без повторного ввода всех параметров.

---

# Функции работы с конфигурациями

Happ предоставляет пользователю набор инструментов для управления профилями:

* создание новых конфигураций подключения;
* импорт настроек из подготовленных файлов;
* изменение параметров существующих профилей;
* хранение нескольких независимых наборов настроек;
* настройка правил маршрутизации;
* редактирование дополнительных параметров соединения;
* выбор используемого профиля;
* обновление сохранённых конфигураций;
* управление всеми настройками непосредственно на устройстве.

Каждый профиль сохраняется отдельно, благодаря чему пользователь может создавать различные варианты конфигураций для разных задач и изменять один набор параметров без воздействия на остальные сохранённые данные.

# Хранение и обработка пользовательской информации

Все созданные профили и импортированные конфигурационные материалы сохраняются только в локальной памяти устройства.

Happ не использует облачные хранилища для размещения пользовательских настроек, не выполняет автоматическую синхронизацию между устройствами и не отправляет данные профилей сторонним сервисам.

Все операции с конфигурациями выполняются самим пользователем через интерфейс приложения. Доступны создание профилей, загрузка файлов, редактирование параметров, сохранение изменений, удаление настроек и активация выбранных вариантов подключения.

Импортированные конфигурационные файлы применяются исключительно внутри Happ и не передаются третьим лицам, организациям или внешним платформам.

---

# Принцип работы приложения

Happ является локальным менеджером пользовательских конфигураций и не предоставляет собственные сетевые услуги.

В состав приложения не входят:

* собственные серверные ресурсы;
* VPN-сервисы;
* прокси-инфраструктура;
* удалённые системы обработки данных;
* встроенные каталоги готовых профилей;
* автоматические механизмы получения или распространения настроек.

Все параметры, необходимые для использования профилей, пользователь добавляет самостоятельно — вручную либо через импорт заранее подготовленных конфигурационных файлов.

Happ не определяет цели применения созданных профилей и не ограничивает варианты их использования. Ответственность за использование приложения и сформированных конфигураций пользователь несёт самостоятельно в соответствии с законодательством и требованиями своего региона.
